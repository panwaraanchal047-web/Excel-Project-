# Excel-Project- Blinkit Sales Analysis Dashboard (Excel Project)
📖 Project Overview

This project presents an interactive Sales Analysis Dashboard built in Microsoft Excel using Blinkit retail data.

The goal of this project is to transform raw sales data into meaningful business insights through data cleaning, analysis, and visualization techniques.

It demonstrates practical skills in data analytics, dashboard design, and business intelligence using Excel.

🎯 Objectives

Analyze total sales performance

Identify top-performing product categories

Compare sales across outlet sizes and locations

Understand customer rating impact on sales

Build an interactive dashboard for decision-making

🛠 Tools & Skills Used

Microsoft Excel

Pivot Tables & Pivot Charts

Slicers for interactivity

Data Cleaning & Transformation

Conditional Formatting

Excel Functions:

SUMIFS

COUNTIFS

IF

VLOOKUP / XLOOKUP

AVERAGE

📊 Dashboard Features

📌 Total Sales KPI

📌 Average Sales KPI

📌 Number of Items Sold

📌 Sales by Item Category

📌 Sales by Outlet Size

📌 Sales by Outlet Location Type

📌 Outlet Establishment Year Analysis

📌 Interactive Filters using Slicers

📂 Dataset Description

The dataset contains the following attributes:

Item Identifier

Item Type

Outlet Identifier

Outlet Size

Outlet Location Type

Sales

Rating

Outlet Establishment Year

📈 Key Insights

Medium-sized outlets generate the highest overall sales.

Urban outlet locations show stronger performance.

Certain product categories contribute significantly to total revenue.

Older outlets tend to have better average sales performance.
